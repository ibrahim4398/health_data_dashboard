## Overview

The **Health Monitoring Dashboard** is a Python-based data analysis and visualization project developed as part of a technical interview task.  
The project demonstrates the complete workflow of handling real-world health data, from raw data retrieval to machine learning classification and interactive visualization.

The main objective is to extract meaningful insights from health metrics and present them in an intuitive and user-friendly dashboard.

---

## Project Scope

This project focuses on the following key areas:

- Health data retrieval
- Data cleaning and preprocessing
- Rule-based health evaluation
- Machine learning classification
- Interactive data visualization

Each stage of the pipeline is implemented using industry-standard Python libraries.
